# Pulse Appoint 🩺📅

A modern appointment booking system designed to simplify scheduling between users and service providers. This project aims to provide a smooth, responsive, and efficient experience for managing appointments.

---

## 🚀 Features

* 📆 Book and manage appointments
* 👤 User authentication (Login/Signup)
* 📊 Dashboard for tracking bookings
* 🔔 Notifications & status updates
* 📱 Responsive UI for all devices

---

## 🛠️ Tech Stack

**Frontend:**

* HTML, CSS, JavaScript, React 

**Backend:**

* Node.js, Express

**Database:**

* MongoDB

---

---

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/haricharanteja-11/Pulse-Appoint.git
cd Pulse-Appoint
```

2. Install dependencies

```
npm install
```

3. Run the project

```
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add:

```
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```
